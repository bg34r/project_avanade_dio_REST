# Projeto Avanade DIO

Projeto prático feito durante o Bootcamp AVANADE pela DIO utilizando Spring BOOT e API Rest.

### Foram utilizados:
* Java 11 ou superior.
* Maven 3.6.3 ou superior.
* intellj IDEA Community Edition
* Git/Github
* Heroku
* POSTMAN
